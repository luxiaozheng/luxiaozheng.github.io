# Welcome to My Blog

This is an example blog post in English. You can write your blog posts in Markdown format and they will be automatically rendered on the website.

## Features

- **Markdown Support**: Write your posts in Markdown for easy formatting
- **Bilingual**: Support for both English and Chinese posts
- **Clean Design**: Beautiful, responsive design that matches your personal website

## How to Add a New Post

1. Create a new `.md` file in the `posts/` directory
2. Add the post information to the `blogPosts` array in `assets/js/blog.js`
3. The post will automatically appear on your blog section
